# PowerBI-Financial-Performance-Dashboard
An interactive Power BI dashboard designed to monitor production costs, material waste analysis, and profitability KPIs for caravan manufacturing operations.
(Portfolio Structure):
1. ملخص المشروع (Project Overview):
الهدف: تطوير نظام تحليل بيانات تفاعلي (Dashboard) لمراقبة تكاليف الإنتاج، حساب نسبة الهدر في المواد الخام، وقياس الربحية التقديرية.

الأداة المستخدمة: Microsoft Power BI.

2. المهارات والتقنيات (Skills & Technologies):
Data Preparation: معالجة وتحويل بيانات الـ CSV لاستخراج رؤى مالية.

DAX Measures: إنشاء مقاييس (Measures) لحساب صافي الربح التقديري، نسبة الهدر، ومتوسط تكلفة الوحدة.

Data Visualization: تصميم تقارير تفاعلية تشمل (KPIs, Combo Charts, Tables) لدعم اتخاذ القرار الإداري.

Business Intelligence: ربط بيانات التكلفة والعمالة لتحليل كفاءة الإنتاج.

3. حل المشكلات (The Value Add):
"تم بناء الداش بورد لتوفير رؤية لحظية حول التكاليف التشغيلية (378K مواد، 100K عمالة)."

"المساهمة في اتخاذ قرارات مدروسة حول خفض الهدر من خلال تحليل العلاقة بين نوع الكرفان وتكلفة الهدر."

"تقديم مؤشرات أداء مالية (KPIs) تساعد الإدارة في تقدير الميزانية وتحديد مدى ربحية المنتجات."
